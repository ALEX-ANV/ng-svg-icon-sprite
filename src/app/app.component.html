{{ title }} (package development app)

Angular package that provides both a solution for generating sprites and a component for including them.

  • ng-svg-icon-sprite sources on Github
  • ng-svg-icon-sprite package on npm

Usage

install ng-svg-icon-sprite and import the npm module via import { SvgIconSpriteModule } from 'ng-svg-icon-sprite'; and add it in any @NgModule imports array.

run npm run create-icon-sprite to create a sprite out of your SVG icons. Include the svg-icon-sprite directive like shown below.
