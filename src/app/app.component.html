{{ title }} (package development app)

Angular package format that provides both a solution for generating sprites and a component for including them.

ng-svg-icon-sprite source on Github

Usage

run npm run create-icon-sprite to create a sprite out of your SVG icons, then run npm run packagr to create the svg-icon-sprite package that will appear in the dist folder.
